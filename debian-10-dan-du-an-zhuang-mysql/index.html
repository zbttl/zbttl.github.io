版本选择

我闲的无聊折腾 mysql 安装就是因为我先装了 8.0 版本在 1g 的 azure 机子上，发现一重启探针就报警吃了 88% 内存，一排查发现是 mysql 的问题。

实测 8.0 版本吃的内存比 5.7 多一倍。。。换为 5.7，内存占用降为 63%。
